K.I.S.S 原则 · 纯本地 · 零数据库

回归纯粹的
极简本地 Agent.

基于 pi-mono 底座。它被严格限制在指定目录之中，以纯文本格式读写记忆。没有数据库的黑魔法，只有你能完全掌控的代码。

bash - stupidClaw

核心哲学

🪶

极低内存占用

抛弃 Redis 和各类向量数据库。仅依赖 Node.js 运行时，几十 MB 内存即可常驻后台，即使在吃灰的树莓派上也能如丝般顺滑运行。

🧩

渐进式技能披露

告别 Token 爆炸。大模型不再被强行塞入上百个工具，而是先查询本地工具目录，再动态按需加载（Lazy Loading）工具上下文，显著降低幻觉率。

⚡️

StupidIM

想快速接入对话？内置 WebSocket 极简通信层，启动即附带本地测试页，聊爽了再一键切换至 Telegram。

🛡️

防呆安全沙盒

我们没有复杂的虚拟机隔离，而是采用"路径锁 (Path Jailing)"。赋予 AI 读写本地文件的能力，但将其物理锁定在 .stupidClaw/ 工作区内，防止大模型删库跑路。

📝

透明的纯文本记忆

没有黑盒。AI 的长期记忆和用户画像全部以 profile.md 和 JSONL 格式落盘保存。你可以随时使用 VS Code 查阅甚至直接手动篡改大模型的记忆。

⏰

主动式 Cron 生物钟

不再"拨一拨动一动"。通过极简的 cron_jobs.json，AI 可以每天早上按时醒来，自主调用搜素技能阅读新闻，并主动通过 IM 推送给你。
